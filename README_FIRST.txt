Visual Studio Code 실행 후 터미널(콘솔창) 띄우기.
1. Server 폴더에 npm install express cors mysql nodemon bcrypt cookie-parser jsonwebtoken multer path
설치

2. frontend 폴더에서는 npm install react-router-dom axios bootstrap bootstrap-icons vite 설치 후 npm init -y


3. frontend 폴더 내에서 npm run dev를 실행시켜 프론트엔드 페이지 로드.

4. Server 폴더 내에서는 npm start로 백엔드 서버 실행

5. xampp 설치 후 어플리케이션 내의 Apache Server와 mysql server 실행
웹 브라우저 실행 -> 주소창에 localhost 입력 -> phpMyAdmin 클릭 -> 'employeems'이라는 이름의 DB생성 -> 'admin'이라는 이름의 테이블 생성 ->
